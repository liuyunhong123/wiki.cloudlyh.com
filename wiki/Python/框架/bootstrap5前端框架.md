# bootstrap5前端框架
用 Bootstrap 构建快速、响应式布局的网站，强大、可扩展、功能丰富的前端开发工具包。它提供的功能包括：基于 Sass 进行编译和定制、内置的网格系统和组件、强大的 JavaScript 插件等。
