## 1. 创建和激活Python虚拟环境
### 创建虚拟环境
```
conda env list
```
### 激活环境
```
conda activate mydjango
```

## 2. 命令行新建django工程Project
### 切换目录，项目存放的目录
```
cd D:\DjangoWork
```
### 新建项目，项目名可以自己更改
```
django-admin startproject antproject
```
