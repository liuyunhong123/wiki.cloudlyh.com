# 登录加验证码
https://django-simple-captcha.readthedocs.io/en/latest/usage.html

插件：django-simple-captcha

## 1、安装命令
```
pip install django-simple-captcha
```

## 2、添加到INSTALLED_APPS中
```
captcha
```

## 3、
```
python manage.py migrate
```

## 4、urls.py
```
urlpatterns += [
path('captcha/', include('captcha.urls')),
]
```
