# 设计数据库表，新建数据model和字段
## 用户表User设计
```
id：自增ID，用户ID
username：用户名，最多字符串120个字符
password：密码，最多60个字符
gender：性别，数字0和数字1，只能是男、女两个选项
role：角色，字符串最多20个字符，admin是管理员，normal是普通用户
create_time：创建时间，datetime形式
```
