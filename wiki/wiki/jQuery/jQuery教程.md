# jQuery
- jQuery 是一个 JavaScript 库。
- jQuery 极大地简化了 JavaScript 编程。
- jQuery 很容易学习。
实例：
```js
$(document).ready(function(){
  $("p").click(function(){
    $(this).hide();
  });
});
```
